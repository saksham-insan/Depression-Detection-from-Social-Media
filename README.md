## 📌 Project Overview

Depression is one of the most common mental health disorders affecting millions of people worldwide. Individuals often express their emotions, thoughts, and feelings on social media platforms, making them a valuable source for identifying signs of depression.

This project uses **Natural Language Processing (NLP)** and **Machine Learning** techniques to classify social media posts as **Depressed** or **Not Depressed** based on textual content.

The goal is to demonstrate how AI can assist in early detection of mental health concerns through automated text analysis.

> **Note:** This project is intended for educational and research purposes only. It is **not a medical diagnostic tool**.

---

# 🎯 Objectives

- Detect depression-related text from social media posts.
- Perform text preprocessing using NLP techniques.
- Train machine learning models for binary text classification.
- Evaluate model performance using standard metrics.
- Compare different algorithms to identify the best-performing model.
